PDF를 올리고, 진행도를 보면서, 번역된 마크다운을 바로 받는 Python SSR 화면

FastAPI 템플릿으로 서버 렌더링하고, 백그라운드에서 PDF를 파싱한 뒤 LangChain으로 번역합니다. 완료되면 MD 파일을 즉시 내려받을 수 있습니다.

Flow

  1. 1. PDF 업로드
  2. 2. 텍스트 추출 및 청크 분리
  3. 3. LangChain 번역 실행
  4. 4. 진행도 polling 표시
  5. 5. 번역된 MD 다운로드
