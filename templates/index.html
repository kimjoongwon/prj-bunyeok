BUNYEOK

PDF를 올리고, 페이지별로 번역하고, 폴더에 페이지 단위 Markdown 파일을 저장하는 Python SSR 화면

FastAPI 템플릿으로 서버 렌더링하고, 백그라운드에서 PDF를 페이지별로 파싱한 뒤 LangChain으로 번역합니다. 화면에 입력한 OpenAI API Key로 요청을 보내고, 완료되면 각 페이지가 별도 Markdown 파일로 저장됩니다.

Flow

  1. 1. PDF 업로드
  2. 2. 페이지별 텍스트 추출
  3. 3. 페이지 단위 LangChain 번역
  4. 4. 진행도 polling 표시
  5. 5. `outputs/...` 폴더에 페이지별 MD 저장
